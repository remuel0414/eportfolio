My Projects
DLSL ISESO Plate Detector
Automatic Plate Number detection for De La Salle Lipa security office, this system automatically detects and saves plate number of cars going in the campus recording it together with other information. There is a dashboard for the data list, and this system also performs data analysis on the said data providing traffic pattern and information. Model was trained using python and the website through flask framework.
Summit Golf Inventory Manager
Inventory Manager of golf, carts and customer items for Summit Point Golf and Residential. The system records rented items, brought-in items, rented golf carts, and customer payment status. Website was created using html, css and bootstrap incorporated with backend database capabilities using php and mysql.
DLSL Grading System
Web based grading system for De La Salle Lipa, users are able to add, delete and search for students with different types of information i.e. student no, name, subject. Automatic computation of grades after input. Website was created using html,css together with php and mysql database record keeping.
